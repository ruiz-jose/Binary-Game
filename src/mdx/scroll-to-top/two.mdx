# Scroll to Top Two
	


## Really Important

Here you can navigate <a style="color:red" href="javascript:history.back()">back to the previous page</a>. This
will not take you back to the end of the page, but to the beginning. This is how it works:

```javascript title="scroll-to-top.js"
document.addEventListener('astro:after-swap', afterSwap);

function afterSwap() {
  window.scrollTo({ left: 0, top: 0, behavior: 'instant' });
}
```
